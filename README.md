# Project starter to run and debug your typescript scripts

How to set up your scripting workspace for TypeScript (tsconfig.json, package.json and Visual Studio Code) and a project starter to run and debug your scripts.

[Documentation about this repository](https://simply-how.com/scripting-with-typescript-dev-tools-configuration)

## Installation

You can install the development dependencies with: `npm install`

## Usage

You can open the project with [Visual Studio Code](https://code.visualstudio.com/)
